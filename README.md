# HR-Analytics-power-bi

HR Analytics Dashboard – Employee Attrition & Performance Analysis

Project Objectives
The main objective of this project is to help an organization improve employee performance and employee retention by analyzing HR data and visualizing key attrition drivers.

Specific goals include:
Identifying major factors contributing to employee attrition
Understanding workforce trends across different demographics
Supporting HR decision-making through interactive visual insights

This project answers the following critical HR questions:

**Attrition by Education**
Analyze how employee education level impacts attrition rates.
**Attrition by Age Group**
Identify which age groups have the highest employee turnover.
**Attrition by Job Role**
Determine job roles with higher attrition to improve role-specific retention strategies.
**Attrition by Salary Slab**
Understand the relationship between salary ranges and employee attrition.
**Attrition by Years at Company**
Analyze how tenure affects employee retention and identify early-stage attrition risks.
**Attrition by Job Role (Detailed Analysis)**
Drill-down analysis of attrition patterns across different roles and departments.

****Tools & Technologies Used**
Power BI – Data visualization & dashboard creation
Power Query – Data cleaning, transformation & modeling
Excel / CSV Dataset – Source data

**Data Preparation****
Removed duplicates and handled missing values using Power Query
Created calculated columns and measures for attrition analysis
Categorized employees into age groups and salary slabs
Optimized data model for better dashboard performance

********Key Insights.****
Certain job roles show consistently higher attrition rates
Employees in lower salary slabs have higher turnover
Early-tenure employees (0–2 years) show increased attrition risk
Specific age groups contribute more to overall attrition

****Business Impact****
Helps HR teams proactively identify attrition risk areas
Supports strategic workforce planning
Enables data-driven decisions to improve employee satisfaction and retention
